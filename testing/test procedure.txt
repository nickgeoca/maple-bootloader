libmaple-Bootloader Testing
Silabs Board

This test document is intended to test the simple functionality of the bootloader. The bootloader must run the application from RAM and Flash. The testing procedure and pass/fail criteria will be described.  

Material Required:
 * SI3MU1XX Precision32 UDP MCU Card
 * Precision32 IDE
 * Silabs debugger

Prerequisites:
 * Setup config.h. FIRST_STAGE_BOOT will likely needed to be defined if bootloader is not being loaded.
 
 1. Test: Execute code from Flash.
   
 2. Test: Execute code from RAM.
